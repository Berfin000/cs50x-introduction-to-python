# CS50P - Introduction to Programming with Python 🐍

This repository contains my personal solutions for problem sets from Harvard University's **CS50P** (Introduction to Programming with Python) course. 

## 🚀 About the Course
CS50P is a rigorous introduction to programming using Python. It covers fundamental concepts of computer science and software engineering, focusing heavily on clean code, problem-solving, and modern Python practices.

## 📁 Repository Structure
Instead of managing dozens of individual repositories, all problem sets are organized logically into subfolders. Each folder contains the complete source code and necessary tests for that specific challenge.

### Key Concepts Covered:
* **Functions & Variables:** Designing modular and reusable code.
* **Conditionals & Loops:** Handling control flow and complex logic.
* **Exceptions:** Writing robust code with proper error handling.
* **Libraries:** Utilizing built-in and third-party Python packages (like `pytest`, `cowsay`, etc.).
* **Unit Tests:** Writing comprehensive tests using `pytest` to guarantee code reliability.
* **File I/O:** Reading and writing files (`.txt`, `.csv`, `.png`).
* **Regular Expressions:** Pattern matching and data validation using `re`.
* **Object-Oriented Programming (OOP):** Creating custom classes, methods, and managing encapsulation.
